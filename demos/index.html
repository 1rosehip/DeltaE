A Poor Man's Chroma Key With dE76

The following demo implements a real-time chroma key to OK Go's song, "WTF". It includes an annotated source that speaks to fun performance roadblocks and solutions.

Name That Color (PureBASIC)

This repository shows an implementation into PureBASIC. It allows users to convert a color into a name over the command line. As an example, #ff4020 becomes Pomegranate.
